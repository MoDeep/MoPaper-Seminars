### Writing Rules
1. 새로운 논문을 추가할 때는 다른 논문들과 규칙을 맞추어서 작성.
2. 되도록이면 논문의 PDF 파일을 하이퍼링크.
3. Alt Text는 '논문 제목 - 제 1저자' 형식 맞추어 작성.
